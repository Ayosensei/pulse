# SwiftyEx Pulse — Mini App

> A proactive AI market companion built on top of SwiftyEx for SwiftyEx Hackfest 2026 — Builders Track

---

## Overview

SwiftyEx today is a powerful but passive bot — it only responds when you ask. **Pulse** changes that.

Pulse is a Telegram Mini App that turns SwiftyEx into an intelligent companion that watches the market for you, greets you with what matters, and speaks up when you need to act — without you having to ask.

---

## The Problem

The current SwiftyEx bot:
- Has no awareness of market conditions
- Waits for the user to initiate every interaction
- Provides no personalized context or intelligence
- Offers no proactive communication

Users trading real money deserve more than a menu.

---

## The Solution

Pulse adds five layers on top of SwiftyEx:

| Layer | What it does |
|---|---|
| **Personalized Greeting** | When you open the bot, you see an AI-generated summary relevant to your coins |
| **Proactive Rate Alerts** | You set a condition; the bot messages you when it's met |
| **Contextual AI Assistant** | A chat interface that reasons about market context, not just FAQs |
| **Rate Pulse Feed** | A live, intelligent rates screen that tells you what the numbers actually mean |
| **Smart Swap Timing** | An agent that recommends the best swap window — and executes it automatically when your target rate is hit |

---

## Features

### 1. Personalized Greeting
When a user opens SwiftyEx, instead of a generic menu dump, they are greeted with a brief, AI-generated market summary tailored to the coins they hold.

**Example:**
> "Good morning! BTC is up 3.2% since yesterday. The USDT/Naira rate is currently ₦1,610 — slightly better than your last swap. Tap below to act."

---

### 2. Proactive Rate Alerts
Users set conditions inside the Mini App. The backend monitors prices in real time and sends a Telegram message when a condition is triggered.

**Supported condition types:**
- Price above / below a threshold (e.g. "Alert me when BTC drops below ₦85M")
- Rate movement (e.g. "Alert me when USDT/Naira moves more than 2% in a day")
- Inactivity nudge (e.g. "Remind me if I haven't swapped in 2 weeks and the rate is favourable")

---

### 3. Contextual AI Assistant
A conversational interface that understands the user's portfolio context and the current market, giving reasoned answers rather than scripted responses.

**Example queries it handles:**
- *"Should I swap my USDT to Naira now?"*
- *"How has the Naira rate moved this week?"*
- *"What's the best time to buy BTC based on recent trends?"*

The assistant is powered by an LLM (Claude/OpenAI) with live market data injected into its context at query time.

---

### 4. Rate Pulse Feed
A live, visual rates screen that replaces the static "View Rates" menu option with an intelligent feed. Instead of just showing numbers, it tells the user what those numbers mean.

**What it shows:**
- Current rates for all supported pairs (BTC/NGN, USDT/NGN, USD/NGN)
- Direction indicator — is this rate higher or lower than yesterday?
- 7-day trend sparkline per pair
- A simple **Now / Wait / Hot** signal generated by the AI based on recent rate movement

**Example:**
> USDT/NGN — ₦1,610 ↑ 1.2% today
> 🟡 WAIT — Rate is trending upward. 7-day average is ₦1,625. Holding 2–3 days could be worthwhile.

---

### 5. Smart Swap Timing ⚡ *(Ace Feature)*
The feature that turns Pulse from a dashboard into an agent. Instead of just watching rates, Pulse recommends the optimal swap window based on historical patterns — and can execute the swap automatically when the rate hits the user's target.

**How it works:**
1. User opens Smart Swap and selects what they want to swap (e.g. USDT → Naira)
2. Pulse analyzes recent rate history and suggests an optimal target rate and time window
3. User sets a target rate and pre-authorizes the swap
4. When the rate hits the target, Pulse executes the swap via the SwiftyEx test environment and notifies the user

**Example:**
> *"Based on USDT/Naira movement over the last 30 days, rates tend to peak between 2PM–5PM on weekdays. The current rate is ₦1,610 — 1.8% below today's likely peak. Set a target of ₦1,635 and we'll swap for you the moment it hits."*

**Why this is different:**
- Every other feature *shows* information — this one *acts* on it
- It's the difference between a dashboard and an autonomous agent
- Demo moment: "Your bot made a swap for you while you were away"

## Screens

### Screen 1 — Onboarding / Preferences
- Select coins you hold (BTC, USDT, etc.)
- Set base currency (Naira / USD)
- Enable/disable proactive alerts

### Screen 2 — Home (Personalized Greeting)
- AI-generated market summary card
- Quick action buttons (Swap, Check Rate, Set Alert)
- Recent activity snapshot

### Screen 3 — Alert Manager
- List of active alerts
- Create new alert (coin, condition, threshold)
- Alert history (triggered alerts)

### Screen 4 — AI Assistant Chat
- Conversational interface
- Market-aware responses
- Context persists within session

### Screen 5 — Rate Pulse Feed
- Live rates for all supported pairs
- Direction indicators and 7-day sparklines
- AI-generated Now / Wait / Hot signal per pair
- One-tap shortcut to Smart Swap from any pair

### Screen 6 — Smart Swap Timing
- Select swap pair (e.g. USDT → NGN)
- AI recommendation card with optimal window and target rate
- Set target rate + pre-authorize swap
- Active smart swap status tracker
- Completion notification history

### Screen 7 — Assets
- Total portfolio balance card with 24h change
- Individual wallet cards for BTC, USDT, USD, Naira
- Each card shows coin balance, NGN equivalent, 24h change badge
- Send and Receive action buttons per wallet

### Screen 8 — Profile & Settings
- Profile section: avatar, display name, Telegram handle, Edit Profile button
- Preferences: toggles for Proactive Alerts, Smart Swap Notifications, Rate Pulse Updates
- Account: KYC Status, Referral Code, Linked Telegram Account
- App: Appearance, Help & Support, Sign Out

### Screen 9 — Bottom Navigation Bar
- Five tabs: Home, Assets, Rates, Swap, Profile
- Active tab highlighted in purple with dot indicator
- Inactive tabs in muted grey
- Persistent across all main screens

## API Contracts

### `GET /greeting`
Returns a personalized AI-generated greeting for the user.

**Request:**
```json
{
  "user_id": "telegram_user_id",
  "coins": ["BTC", "USDT"],
  "base_currency": "NGN"
}
```

**Response:**
```json
{
  "greeting": "Good morning! BTC is up 3.2% today...",
  "rates": {
    "BTC_NGN": 85000000,
    "USDT_NGN": 1610
  },
  "last_updated": "2026-06-06T08:00:00Z"
}
```

---

### `POST /alerts`
Create a new proactive alert.

**Request:**
```json
{
  "user_id": "telegram_user_id",
  "coin": "USDT",
  "condition": "above",
  "threshold": 1650,
  "currency": "NGN"
}
```

**Response:**
```json
{
  "alert_id": "abc123",
  "status": "active",
  "message": "Alert set. We'll notify you when USDT/NGN goes above ₦1,650."
}
```

---

### `POST /chat`
Send a message to the AI assistant.

**Request:**
```json
{
  "user_id": "telegram_user_id",
  "message": "Should I swap my USDT to Naira now?",
  "context": {
    "coins": ["USDT"],
    "last_swap": "2026-05-20",
    "current_rates": { "USDT_NGN": 1610 }
  }
}
```

**Response:**
```json
{
  "reply": "The USDT/Naira rate is currently ₦1,610, which is slightly below the 7-day average of ₦1,625. If you're not in a rush, waiting a few days could yield a marginally better rate. That said, if you need the Naira now, today's rate is still reasonable."
}
```

---

### `POST /smart-swap`
Create a pre-authorized smart swap that executes when target rate is hit.

**Request:**
```json
{
  "user_id": "telegram_user_id",
  "from_coin": "USDT",
  "to_coin": "NGN",
  "amount": 100,
  "target_rate": 1635,
  "expiry": "2026-06-07T23:59:00Z"
}
```

**Response:**
```json
{
  "swap_id": "swp_xyz789",
  "status": "watching",
  "target_rate": 1635,
  "current_rate": 1610,
  "recommendation": "Based on 30-day patterns, rates tend to peak between 2PM–5PM on weekdays. Your target of ₦1,635 has a good chance of being hit within 24–48 hours.",
  "message": "Smart swap armed. We'll execute automatically when USDT/NGN hits ₦1,635."
}
```
